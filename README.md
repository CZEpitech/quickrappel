# QuickRappel

A tiny macOS desktop panel for Apple Reminders. It sits on your desktop like a widget, shows your open reminders, and lets you add new ones by typing directly into it - no app to open, no extra window.

Native WidgetKit widgets cannot contain text input (Apple limitation), so QuickRappel renders its own always-visible panel at desktop level instead: it stays behind your windows, on every Space, exactly where you put it.

## Features

- Live list of incomplete reminders from Apple Reminders (iCloud sync: iPhone, iPad, other Macs)
- Inline text field: type, press Enter, done
- Click the circle to complete a task
- Multi-level undo (Cmd+Z or the arrow button, up to 20 actions): un-complete a task, remove a just-added reminder
- Natural language dates: `call the dentist tomorrow 2pm`
- Recurring tasks with `!`
- Menu bar icon: show/hide panel, lock position, language, quit
- Right-click the panel: lock its position so it cannot be dragged, switch language
- French and English UI
- Launches at login, remembers its position

## Input syntax

| You type | Result |
|---|---|
| `buy milk` | due today |
| `buy milk tomorrow 2pm` | due date with alarm |
| `!work out` | repeats every day |
| `!work out 6pm` | repeats every day at 6pm |
| `!monday take out the trash` | repeats every Monday |
| `!monday,thursday gym 6pm` | repeats Monday and Thursday at 6pm |

Weekday names work in English (`monday`...`sunday`) and French (`lundi`...`dimanche`), accent-insensitive.

## Requirements

- macOS 14 (Sonoma) or later
- Xcode command line tools
- [XcodeGen](https://github.com/yonaskolb/XcodeGen) (`brew install xcodegen`)
- An Apple Development signing certificate (a free Apple ID added in Xcode is enough)

## Build and install

```sh
git clone https://github.com/CZEpitech/quickrappel.git
cd quickrappel
xcodegen generate
xcodebuild -project QuickRappel.xcodeproj -scheme QuickRappel -configuration Release -derivedDataPath .build build -allowProvisioningUpdates
ditto .build/Build/Products/Release/QuickRappel.app /Applications/QuickRappel.app
open /Applications/QuickRappel.app
```

Set `DEVELOPMENT_TEAM` in `project.yml` to your own team ID before building.

On first save, macOS asks for permission to access Reminders - allow it.

To start the app at login, add it in System Settings > General > Login Items, or:

```sh
osascript -e 'tell application "System Events" to make login item at end with properties {path:"/Applications/QuickRappel.app", hidden:false}'
```

## Regenerating the app icon

The icon is generated by a script:

```sh
swift tools/gen_icon.swift /tmp/icon_1024.png
```

Then convert it to `Resources/AppIcon.icns` with `sips` and `iconutil`.

## Privacy

QuickRappel talks only to the local EventKit store (Apple Reminders). No network calls, no analytics, no data collection.
